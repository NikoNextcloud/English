# WordJoy English

MVP уеб приложение за обучение по английски език с дневна цел от 10 думи. Проектът е статичен и може да се качи директно в GitHub Pages.

## Какво включва

- Регистрация с имейл или демо Google вход
- Избор на ниво: A1, A2, B1-B2, C1-C2
- Начален екран със streak, XP, ниво и прогрес
- 100 урока по 10 думи, общо 1000 думи
- Произношение чрез Web Speech API
- Микрофон проверка, когато браузърът я поддържа
- Упражнения: превод, подреждане, избор на отговор и слушане
- Граматични модули
- XP система и потребителски нива
- Spaced repetition: 1, 3, 7 и 30 дни
- Локален AI teacher симулатор
- Статистика и модел за бъдеща база данни

## Стартиране

Отвори `index.html` в браузър.

За GitHub Pages качи папката `english-learning-a2-c1` в repository и избери Pages deployment от branch.

## Следващи стъпки за реален продукт

- Свързване с Firebase или Supabase за Users, Lessons, Words и Progress
- Реален Google OAuth
- Реален AI чат през API
- Истински аудио файлове за думите
- Админ панел за добавяне на думи и уроци
