# LinguaLeaf — изцяло статично приложение

Преработена версия на приложението за учене на английски. Вече е **100% статично** —
няма Node.js сървър, няма AI бекенд, нищо не се изпраща навън. Всичко работи в браузъра.

## Файлове

- `index.html` — входна точка
- `app.js` — цялата логика (без промени по `data.js`)
- `data.js` — 100 урока × 10 думи (4 нива: A1, A2, B1, C1)
- `styles.css` — новият дизайн, респонсив
- `assets/teacher-hero.jpg` — оптимизирана картинка (1.6 MB → 80 KB)

## Как да го пуснеш

Просто отвори `index.html` в браузъра. Прогресът се пази локално (localStorage).

Или с локален сървър (по желание):

```bash
npx serve .
# или
python3 -m http.server 8000
```

## Качване в интернет (GitHub Pages)

1. Качи всички файлове в GitHub repo.
2. Settings → Pages → Source: `main` branch, root.
3. Готово — линкът се появява след минута.

Работи и на Vercel / Netlify чрез drag-and-drop на папката, без настройки.

## Какво е премахнато

- AI чатът с учителя и целият `server.mjs` бекенд (OpenAI).
- `package.json`, `render.yaml` — вече не са нужни.

Упражненията за писане останаха, но се оценяват **офлайн** в браузъра.
